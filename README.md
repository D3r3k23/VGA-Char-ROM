# VGA-Char-ROM

A lab project for my digital systems class. The project was designed to work on a Blackboard from [Real Digital](https://www.realdigital.org/), which includes a Znyq SoC with an ARM processor and Artix FPGA. The project includes a VGA controller which can draw a single character on the display, written in Verilog. The character, its location, and color is controlled by software using the AXI bus to communicate with the ARM
